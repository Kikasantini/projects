Scripting with Python

Python3 | Shell | Linux

This was a school project where we had to automate daily activities of a fictitious enterprise. So I chose a dummy translation company and implemented 3 scripts to streamline the processes.

My first script calculates the price of each file to be translated based on the translation type (simple or sworn) and the number of pages, then adds each amount to a text file. The second script adds an 'official' registry number for each page of the translated file, and an extra page with the translators' signatures. Finally, the third script calculates and records the total revenue for each day of work.

For this project, I used the following packages and resources: Python3, Python Pip, PyPDF2, ReportLab and Crontab.

Mobile game

C# | Unity | Visual Studio | Photoshop

Gaming has been an important part of my life since I can remember. And I like all sorts of games. I started this project in 2020 with a clear objective in mind: combine all good things I've experienced from many years of gaming in 1 single mobile game. Easy peasy, right?! The idea here is actually to combine different mechanics, and make it so you need to play all of them in order to improve.

This has been a challenging and amazing experience, and there's so much more to learn and implement. I originally used free sprites/images from the internet, but I've been making my own sprites and I'm replacing the old ones gradually.
Here you can see the old sprites, but I'll show you the ones I made in the next section of the portfolio.

You can also download my game's APK >> HERE << to install on Android. It works for aspect ratio 20/9, I haven't done the UI responsive yet. By the way, the game is not finished or balanced, and the save system only works on PC build for now, but I'm getting there!
