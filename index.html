Scripting with Python

Python3 | Shell | Linux

This was a school project where we had to automate daily activities of a fictitious enterprise. So I chose a dummy translation company and implemented 3 scripts to streamline the processes.

My first script calculates the price of each file to be translated based on the translation type (simple or sworn) and the number of pages, then adds each amount to a text file.

The second script adds an 'official' registry number for each page of the translated file, and an extra page with the translators' signatures.

Finally, the third script calculates and records the total revenue for each day of work.

For this project, I used the following packages and resources: Python3, Python Pip, PyPDF2, ReportLab and Crontab.

Mobile game

C# | Unity | Visual Studio | Photoshop

Gaming has been an important part of my life since I can remember. I started this project in 2020 with a clear objective in mind: combine all good things I've experienced from many years of gaming in 1 single mobile game. Easy peasy, right?!

This has been a challenging and amazing experience, and there's so much more to learn and implement. This project is still ongoing and all the sprites I used are free.

About the game : it is a turn-based RPG. There are 3 characters, each one with 3 different skills and equipments. The player must farm coins and skills and upgrade the equipment in order to defeat the boss and level up.
