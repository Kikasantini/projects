Scripting - School Project
Python3 | Shell | Linux

This was a school project where we had to automate daily activities of a fictitious enterprise. So I chose a dummy translation company and implemented 3 simple scripts to streamline the processes.

First Script : prix.py
Calculates the price of each file to be translated based on the translation type (simple or sworn). Then adds each amount to a text file.

Second Script : registre.py
For each translated file, adds a registry (like an official registry number) to each page, and a final page with the translators' signatures.

Third Script : somme_revenu.py
Calculates and records the total revenue for each day of work.

For this project, I used the following packages and resources: Python3, Python Pip, PyPDF2, ReportLab and Crontab.

Also, this was my first contact ever with Python. And I enjoyed it so much that it instantly became one of my main interests that I'd love to keep exploring professionally.
